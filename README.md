# FarmRush - Neon Cyber Mining

A high-performance Telegram Mini App (TMA) built with Next.js, Genkit, and Firebase.

## Features

- **Energy-Core System**: 1000/1000 capacity with full-recharge rewarded ad mechanism.
- **USDT Payout Economy**: Sustainable 200k:1 USDT conversion with tiered withdrawals and dynamic cooldowns.
- **VIP Earning Protocol**: Tiered investment plans (Silver, Gold, Diamond) with manual Crypto verification (BEP20).
- **Ad-Economy Hub**: Multiple earning streams including Cinema Room, Video Chests, and Ad Streaks.
- **Network Milestones**: High-reward referral ladder for "Verified" friends (3 to 100 referrals).
- **Admin Control Hub**: Real-time management of VIP requests and withdrawal ledger.
- **AI Strategy Engine**: Genkit-powered crop and mining optimization.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS + ShadCN UI
- **Backend**: Firebase Firestore & Auth
- **AI**: Genkit 1.x (Google Gemini)
- **Icons**: Lucide React

## Getting Started

1. Set up your Firebase project in `src/firebase/config.ts`.
2. Run `npm install`.
3. Launch development server: `npm run dev`.
4. Access the Admin Panel at `/admin` (Default Key: `admin123`).
